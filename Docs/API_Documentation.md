# Backend Schema

The backend schema defines the structure of user records in the system.

```json
{
  "medical_record_number": "string",
  "age": number,
  "sex": "M" or "F" : {enum},
  "pro_nouns": "string",
  "zip_code": "string",
  "latest_bmi": "string",
  "latest_weight": "string",
  "png_filename": "string",
  "exam_id": "string",
  "icu_admit": "Y" or "N" : {enum},
  "icu_admits_count": number ,
  "mortality": "Y" or "N" : {enum}
}
```
